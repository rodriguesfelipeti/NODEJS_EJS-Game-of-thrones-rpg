# mmorpg-Game-of-Thrones-NodeJS-MongoDB-EJs-
mmoRPG Game of Thrones (NodeJS + MongoDB + Ejs)

Sistema de estudo NodeJS + MongoDB + Ejs.
Rpg, com cadastro e login de usuários, eventos de jogador, usando BD Mongo.
Inclusão de crypto, para criptografia de senhas.
